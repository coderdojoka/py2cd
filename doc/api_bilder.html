Bilder¶

Bilder können über den BildSpeicher geladen werden. Dieser verwaltet alle Bilder und stellt diese über einen frei wählbaren Schlüssel zur Verfügung.

Lädt das Bild aus der beschrieben Datei. ACHTUNG: Kann das Bild nicht geladen werden, wird ein Fehler geworfen!
