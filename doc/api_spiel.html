Spiel - Die Hauptklasse¶

Die Hauptfunktionen von py2cd sind hier gebündelt. Spiel.init() und Spiel.starten() sind die beiden wichtigsten Funktionen.
class py2cd.spiel.Spiel¶

Die Hauptklasse des Spiels. Es muss Spiel.init() und Spiel.starten() aufgerufen werden.

classmethod init(breite=640, hoehe=480, titel='Py2cd Zeichenbiblothek', aktualisierungs_funktion=<function Spiel.<lambda>>)¶

Initialisiert das Spiel.

HINWEIS: Diese Funktion muss aufgerufen werden, bevor py2cd Funktionen verwendet werden können.

Parameter:
  • breite (int) – die Fensterbreite
  • hoehe (int) – die Fensterhöhe
  • titel (str) – Der Fenstertitel
  • aktualisierungs_funktion ((float) -> None) – die Aktualisierungsfunktion, die bei jedem Neuzeichnen aufgerufen wird (fps mal pro sekunde)
static registriere_maus_bewegt(funktion)¶

Registriert eine Funktion, die ausgeführt wird, wenn eine Maus-Taste gedrückt wird. Wenn die Funktion aufgerufen wird, wird ihr ein Objekt übergeben, das so aufgebaut ist:

button: 1-3 # welche Taste: 1 = Links, ... pos: (x,y) # Tupel mit der Position
Parameter:funktion ((object) -> None) – Die Funktion die aufgerufen werden soll, wenn eine Taste gedrückt wurde
static registriere_maus_gedrueckt(funktion)¶

Registriert eine Funktion, die aufgerufen wird, wenn eine Maustaste gedrückt wird.

Parameter:funktion ((object)->None) –
static registriere_maus_losgelassen(funktion)¶

Registriert eine Funktion, die aufgerufen wird, wenn eine Maustaste losgelassen wird.

Parameter:funktion ((object)->None) – Die Funktion
static registriere_taste_gedrueckt(taste, funktion)¶

Registriert eine Funktion, die ausgeführt wird, wenn die angegebene Taste gedrückt wird.

Parameter:
  • taste (int) – die Taste z.B. die a-Taste ist 97. Alle Tasten sind vordefiniert, so entspricht K_a der ‘a’-Taste
  • funktion – Die Funktion die aufgerufen wird. Sie muss 2 Parameter akzeptieren, der

Erste gibt an, ob die Taste gedrückt oder losgelassen ist und der Zweite ist das Event Objekt :type funktion: (bool, object) -> None
